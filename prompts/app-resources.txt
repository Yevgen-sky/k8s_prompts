Role: Senior Kubernetes Performance Engineer.

Generate a Pod manifest with resource requests and limits.

- apiVersion: v1
- kind: Pod
- name: app-resource
- image: gcr.io/kuar-demo/kuard-amd64:1

- resources:
  requests:
    cpu: 100m
    memory: 128Mi
  limits:
    cpu: 100m
    memory: 256Mi

CONSTRAINTS:
- Produce valid YAML
- No comments or text

FORMAT:
Strict YAML

STEPS:
1. Define metadata.
2. Add container.
3. Add probes.
4. Add resource section.
5. Add port.
6. Output YAML.
